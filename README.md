# toolsGoods

香水商品运营平台仓库，当前围绕四件事做商品决策：

- 防止好卖商品断货
- 识别临期和过期风险批次
- 找出长期不动销的滞销商品
- 给出门店之间的调拨建议

## 当前进度

第二版已经开始接真实业务文件：

- 支持模板明细表导入
- 支持 `90天销售.csv` 这类订单商品流水自动聚合
- 支持 Excel / CSV 导入
- 支持通用字段映射

## 目前最适合的导入方式

1. 一张标准明细表：一行代表一个 `SKU + 门店 + 批次`
2. 一张 90 天销售流水表：系统会自动聚合成 `SKU + 门店` 销售结果

## 后续待接

- 库存文件
- 在途库存文件
- 保质期 / 批次文件

等这些文件接上后，补货、临期和调拨判断就会完整跑起来。
